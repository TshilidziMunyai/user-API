# user-API

## Description
A REST API to manage user information with basic CRUD operations.

## Features
- `GET /users`: Retrieve all users.
- `POST /users`: Create a new user.
- `PUT /users/{id}`: Update user details.
- `DELETE /users/{id}`: Delete a user.

## Requirements
- Python 3.8+
- Flask
- Flask-SQLAlchemy
